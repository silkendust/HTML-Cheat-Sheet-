Elements

There are many different elements to choose from when writing your html. The following list is a few of the most used for quick reference:

Tag Name Description
<a> Anchor Creates a link to another page or to a location in the current page.
<blockquote> Block Quote Represents a section of a document which contains a longer quotation, usually spanning multiple lines.
<br> Text Break Represents a break in text. It is used when text needs to span multiple lines rather than being in-line, such as in an address.
<tr> Table Row The table row container.
<button> Button Represents a button to be clicked by a user.
<div> Divider Represents a generic division of content. It has no semantic meaning, but will separate its contents from the rest of the document.
<embed> Embed Inserts external content such as a video, image, page, and more.
<head> Head of html Represents a collection of metadata related to the current document. It is an immediate child of the `<html>` element and may include other tags such as <title>, <link>, <style>, and <script>
<img> Image Displays an image on the web page.

Attributes

HTML attributes are data that can be added to the opening tag of an HTML element. These attributes modify the default behavior of the element or provide additional information about it.

Attributes consist of a name and are usually set equal to a value. Values are enclosed in double or single quotation marks.

Tag Name Description
class Class An HTML attribute used to select one or more elements for the application of styling or logic.
dir directions Sets the direction of the text of an element.
id ID Used to specify a unique id for an HTML element, which cannot be shared across multiple elements within the same HTML document.
src Source Specifies the location of a digital source.
style Style Specifies how an element will be rendered in a web browser.
title Title Contains and can display the specified information about the element it belongs to.
